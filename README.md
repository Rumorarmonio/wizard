# Wizard

Bootstrap проекта на Nuxt 4, Vue 3 Composition API и TypeScript для будущего onboarding wizard.

## Команды

```bash
npm install
npm run dev
npm run build
```

## Текущее состояние

- Инициализирован базовый Nuxt 4 проект.
- Подключён глобальный слой стилей через `app/assets/styles/main.scss`.
- Для локальных стилей компонентов и страниц используются отдельные `.module.scss` файлы.
- Бизнес-логика wizard и шаги сценария пока не реализованы.
